# Archive: Pre-Research Sivart Skills

These files are preserved from the original `sivart` repo (commit `c9c5213`),
created before the deep research on the `/goal` primitive, Auftragstaktik,
principal-agent theory, and the Mission Contract philosophy.

They are preserved here as inspiration and as a record of the iterative path.
The evolved system replaces them with the Mission Contract Schema, the
Intent Cascade Linter, and the Mission Orchestrator skill.

## Contents

- `docs/kanban-mission-system.md` — Original concept document describing the
  four-phase workflow (Define, Plan, Validate, Deploy) and "Mission Packages."
- `skills/define-mission/SKILL.md` — The "Strategist" skill for writing
  mission manifests.
- `skills/plan-mission/SKILL.md` — The "Operations Officer" skill for
  generating deployment bash scripts.
- `skills/validate-mission/SKILL.md` — The "Inspector General" skill for
  static bash script validation.

## Status

Superseded. See `../../README.md` for the current architecture and
`../../_template.md` for the Mission Contract Schema.
